# REST API Testing using Postman 🚀

This project is a **Spring Boot REST API application** created for learning and practicing backend development and API testing using Postman.

---

## 📌 Overview

This project demonstrates how to:
- Build RESTful APIs using Spring Boot
- Perform CRUD operations
- Test APIs using Postman
- Understand backend request-response flow

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring Web (Spring MVC)
- Maven
- Postman
- Embedded Tomcat
